# Zaplanuj Jedzonko - plan a meal app.
##Description:

Plan your meal, an application that lets you create recipes and nutrition plans :)

##Technologies:
<ul>
<li>Java 8</li>
<li>Spring</li>
<li>Spring Security</li>
<li>Spring Data</li>
<li>Hibernate</li>
<li>JSP</li>
<li>MySQL</li>
<li>Maven</li>
</ul>
